## ✨ 功能

- **🧪 成分庫** — 內建常見保養品成分資料（A醇、維他命C、玻尿酸、酸類等）
- **⚠️ 衝突檢測** — 自動檢測成分間的配伍禁忌（如 A醇+酸類）
- **☀️🌙 早晚分流** — 區分早/晚/早晚皆可使用的成分
- **📋 流程管理** — 新增、刪除保養品，建立個人化流程
- **🏷️ 預設產品** — 內建常見產品快速選擇
- **💾 本地儲存** — 資料儲存在 localStorage
